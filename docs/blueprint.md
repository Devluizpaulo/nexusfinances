# **App Name**: Nexus Finances

## Core Features:

- Income Tracking: Log income from various sources such as salary, freelance work, or commissions. Supports recurring and one-time entries.
- Expense Tracking: Record both fixed and variable expenses, categorized for easy analysis. Automate recurring entries like subscriptions and bills.
- Debt Management: Manage installment debts by tracking individual installments, payment status, and due dates.
- Automated Recurrence Generation: Automatically generate recurring income and expense entries based on user-defined schedules.
- Installment Plan Generator: Automatically generate installment plans by defining the number of installments, value, due dates.
- Financial Dashboard: Visualize financial data through interactive charts and key performance indicators (KPIs). The dashboard offers insights into income, expenses, debts, and savings.
- Educational Financial Insights: Use an LLM to generate non-intrusive insights based on financial trends, such as increased spending in a specific category or progress towards savings goals. The LLM should decide when to include educational recommendations in its analysis and reporting. This acts as a helpful tool to teach finance concepts

## Style Guidelines:

- Primary color: Calm and sophisticated dark blue (#3B5998), evoking trust and reliability.
- Background color: Soft light grey (#F0F2F5), creating a clean and spacious feel.
- Accent color: Muted purple (#8075FF), used sparingly for highlights and calls to action to maintain a professional appearance.
- Body and headline font: 'Inter', a sans-serif font providing a modern, neutral, and easily readable appearance across all text elements.
- Use clean, minimalist icons to represent financial concepts, reinforcing a modern and user-friendly design.
- Maintain a structured and spacious layout to avoid overwhelming users with financial data. Prioritize clarity and ease of navigation.
- Incorporate subtle transitions and loading animations to improve user experience. Ensure animations are non-distracting and add a layer of polish.
